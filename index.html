Installing the app

Gunak app can be installed from [Google Play](https://play.google.com/store/apps/details?id=com.facilitiesassessment).

Frequently Asked Questions
  • What is assessment number?

    If you are performning assessment of a facility from multiple devices, then from each of the device after selecting the same facility you can choose the same assessment number. The app comes a button which can generate a unique number based on today's date. If all users on their devices use this button on the same day then they can share the same assessment number. Otherwise a group can edit this number manually to after agreeing on a common number.
